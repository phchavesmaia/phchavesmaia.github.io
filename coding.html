Coding and other Resources

This section links to resources that I have found helpful throughout my Ph.D. journey or have made myself. Graduate Students and Research Assistants may find these particularly helpful.

Big Data

dask

Allows you to leverage your pandas (dask.dataframe) knowledge to big data situations.

Spatial Data

GIS Methods Course

Sebastian Hohmann's lectures (for economists) on using Python to work with GIS data. An incredible resource which almost single-handedly taught me how to work with spatial data.

dask-geopandas

Same as dask but for GIS data.

Maps in Stata

Asjad Naqvi's guide on using Stata to generate maps.
